info Информация

Группы пользователей позволяют ограничивать доступ к endpoints.

  • Каждый пользователь может быть только в одной группе
  • Пользователь видит только endpoints, доступные его группе
  • Один endpoint может быть доступен в нескольких группах
  • Группа "Default" создаётся автоматически

После создания группы вы сможете назначать пользователей в неё при создании или редактировании пользователя.
